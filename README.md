# mpi_learn

Learn parallels programming with **C** and MPI, Pthread, OpenMP this three of API. I use MacOS to code while I provide the way to install **mpich** in Ubuntu20.04.

### MacOS

``` bash
$ brew install mpich
```

### Ubuntu20.04

``` bash
$ sudo apt install mpich
```
